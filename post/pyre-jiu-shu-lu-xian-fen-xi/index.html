Pyre救赎路线分析
2021-04-22

相比《Transistor》，PYRE中的人物更多，性格各不相同，感情线更多了，这里做个小记录。
笔者的救赎顺序:
乔达里尔、鸟妹妹(非新夜翼)、Tizo、海蛇、赫得文、大树、恶魔奶爸(非新夜翼)

为什么要回人间？:

Reader：没兴趣、不想回
赫得文：想女朋友了
乔达里尔：回不回都行...
狗哥: 回家找妈妈，还贷款，做生意养家。希望妈妈不要知道自己其实过得很不好...
菲菲（野蛮人女孩，下文统一称为菲菲）：人间也没人要我，在下届还有朋友，有著者,有远古知识。不愿意回人间。
Tizo: 精灵本就该呆在下届，没必要去人间。
海蛇：也许回到上界，就可以夺回尊严和荣耀。 但没必要。
鸟姐: 回去找妹妹，希望妹妹不要过打打杀杀的生活。
大树：生活在水深火热之中的联邦百姓需要解放。
巫婆姐姐: 大树去哪，我去哪

个人事业线：

大树.革命
海蛇.骑士的荣耀
鸟妹: 高翼族复仇，毁灭联邦
巫婆姐姐：成就大树的事业
菲菲：学业

感情线:

赫得文 x 乔德利尔 好丽友
赫得文 x 赫夫人 爱情
大树 x 巫婆姐姐 爱情
大树 x 恶魔奶爸 好丽友
鸟姐 x 鸟妹 好丽友
鸟姐 x 狗狗 爱情
菲菲 x 狗小哥 爱情
菲菲 x Tizo 好丽友
乔达里尔 x 老伊 爱情
感情线总结如下

这里解释一下。

  • 狗狗和赫得文 是有强烈个人意愿 回人间
  • 乔达里尔、鸟妹、恶魔奶爸、大树 对革命帮助大，考虑回人间
  • Tizo和海蛇 纯粹是个人爱好，回不回人间 都行。
  • 盲目姐一定要跟着Reader ，而Reader本身去哪都行
  • 鸟姐 和 狗狗有爱情线，最好在一起；
  • 巫婆姐姐和树人有爱情线，最好在一起。
  • 同时 鸟姐和巫婆还是闺蜜，最好在一起。
  • 菲菲对远古知识有渴望 同时 和老狗之子有爱情线，是有强烈意愿留在下界
  • 老伊 对 乔达里尔有爱情线，最好在一起。
  • 霸克 和 狗狗关系不好，不要放一起。

总结

救赎方案：
  • 革命为第一目标的最优救赎方案为:
    [乔达里尔、恶魔奶爸(非新夜翼)、大树、鸟妹(非新夜翼)、赫得文、鸟姐、巫婆姐姐 ]
  • 情感为第一目标的最优救赎方案为：
    [狗狗、赫得文、鸟姐、鸟妹(非新夜翼)、巫婆姐姐、大树、恶魔奶爸(非新夜翼)]
游戏小技巧:
  • 反马太效应:
    送走的都是熟练度高的角色，所以不要在 计划要救赎的人身上投入太多财力和个人情感。
  • 仪式战略:
    一后卫 x 两个得分
    如: 大树 + Tizo + 鸟姐
    一后卫 x 一个得分 x 一个杀手
    如：乔达里尔 + 狗狗 + 巫婆姐姐

最后的小牢骚：
PYRE在游戏模式 故事剧情 音乐 画面 四个方面都好棒。画风优雅，视频流畅，音乐震撼，剧情新颖，游戏模式有很有创意。
